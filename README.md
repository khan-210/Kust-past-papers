# 📚 University Past Papers Repository

This repository contains past papers for university students. The main purpose of this repo is to help students prepare better for exams by providing previous year papers in one place.

---

## 📌 Subjects Included

You can find past papers of the following subjects:

- Programming Fundamentals
- Object Oriented Programming (OOP)
- Digital Logic Design
- Mathematics
- English
- Database Systems
- Data Structures

(You can add or remove subjects according to your university)

---

## 🎯 Purpose of This Repository

This repository is created to:

- Help university students in exam preparation
- Provide all past papers in one place
- Save students’ time
- Help juniors in future semesters

---

## 📥 How to Use This Repository

1. Open the subject folder
2. Download the past paper
3. Practice it before exams
4. Try to solve without looking at answers

---

## 🤝 Contribution

If you have past papers that are not available in this repository, you can contribute.

You can:
